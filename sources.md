It can be proven that, in fact, there are only two cases: one where the infinite loop converges to 1 or the loop converges to 89: 
[Stack Exchange](https://math.stackexchange.com/questions/1933385/repeated-sum-of-square-of-digits-always-arrives-at-1-or-89)
Therefore, the code can be streamlined to terminate the while statement if 1 or 89 appear at any point in the process. This should reduce the number of needed loops for large *n*.
